Try It Now

  • Docker
  • OpenShift
  • Kubernetes

Run the following Docker command in your terminal:

#: docker run -ti -v /var/run/docker.sock:/var/run/docker.sock -v ${LOCAL+PATH}:/data eclipse/che start

Deploy to Minishift

#: docker run -ti -v /var/run/docker.sock:/var/run/docker.sock -v ${LOCAL+PATH}:/data eclipse/che init #: ${LOCAL+PATH}/instance/deploy_che.sh
